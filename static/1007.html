練習

  1. 多人專案,發生衝突的解決 (not yet💖)
  2. 複製別人的repo 順便練習殼層指令
  3. 先在github上建立自己的repo 例如 u777
  4. 進入codespaces
  5. 在終端機中cd ..
  6. git clone https://github.com/airbone4/hw0930.git
    • 此時,有兩個子目錄,一個是u777一個是hw0930
  7. cd hw0930
  8. rm -rf .git 刪除.git這個子目錄
  9. cd ..
  10. cp hw0930/* u777 此時已完整複製所有對方的檔案到自己的工作區
  11. git add .
  12. git commit -m "任何訊息"

  13. git push
