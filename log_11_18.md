
javascript的字串表示方式有三種
1. '
1. "
1. `

其中反頓號,讓我們直接可以把變數以${變數}的方式代換。
例如

```js
let a="你好"
let b=`目前變數a的內容是${a}`
console.log(b)

```
出現的結果是
```
目前變數a的內容是你好
```

```js
let a="你好"
let b=`目前變數a的內容是${a}`
console.log(b)
a="天氣好"
b=`目前變數a的內容是${a}`
console.log(b)

```
出現的結果是
```
目前變數a的內容是你好
目前變數a的內容是天氣好
```

```js
let a=5
let b=3
let c=a
c=a+b
console.log(c)
```

最後的結果就是
```
8
```


```js
let a=5
let b=3
let c=a
c=a+b
console.log(`a的內容是${a}, b的內容是${b} , c 是a+b的結果,內容是 ${c}`)
```

最後的結果就是
```
8
```

- 數字可以做加減乘除
- 但是不能像一般的程式語言有平方的寫法 ,寫法是 `Math.pow(a,b)` 等於是excel 中 a^b

宣告變數的方法有let ,var ,const

- const 指定的變數值無法修改
- let,var 看起來都一樣 但是let 是新標準的寫法,比較不會出問題
  1. c=4  === var c=4 這時候c是一個數字變數,c的型態(type) 就是數字
  1. c='abc' === var c='abc' 這時候c 的型態是字串。
- 型態還有邏輯值
  `true`,`false` true 是非0,false 是 0  邏輯運算子(operator)有 and (&) or (|) 還有`非` `等不等於`
