```js

function sayHello(msg) {
    let y=4;
    alert(msg+'x: '+x+'y: '+y);
}        


```
- 變數x 是global variable (總體變數,全局變數)
- 變數y 是local varialbe (局部變數) 
  - 局部變數的第一個特徵是,外面有block ,例如`｛`和`｝`之間
- 存取關係:外面拿不到裡面(local variable),裡面可以拿到外面(global variable)  


```js
    <p id="output"> </p>

		    let x=4;  //local variable
			let y=5;  // local variable
			let output= document.getElementById('output');  
            //document 是 global variable
            // output 是local variable

			output.innerHTML = msg+'x: '+x+'y: '+y;

		}        

		sayHello('hello ')
     
    </script>
	<script>
	 console.log(x)  //這裡拿到的是global variable
	</script>
```    

## var 宣告 let 宣告 哪裡不一樣

let 的生命範圍是block ,但是var 的生命範圍 不是block 而是函數block

看下面的程式碼註解
```js
	    // var y=5;  //註解1:
        {// 這是block 但不是函數block
	    let x=3;
		var y=5;  //在這裡宣告,等於在註解1宣告。因為var 宣告的y ,其生命期只受函數block 影響
		}
	 
		console.log(y)
		console.log(x)

        //函數sayHello的定義
		function sayHello(msg) 
        { //這是函數block
		    let x=4;  //函數結束,x消失 (結束生命期)
			var y=5; //和x一樣 也是,函數消失就結束。因為 var 宣告的變數,受函數block影響。
			let output= document.getElementById('output');
			output.innerHTML = msg+'x: '+x+'y: '+y;
		}        

		sayHello('hello ') //調用函數sayHello
```
